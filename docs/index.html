Импорт отгрузок из XLSX / CSV файлов с вашего диска

Загрузить отгрузки (XLSX)

Выберите .xlsx файл с диска. Поддерживаются форматы:
• Один лист на дилера (название листа = имя дилера), строки = категории, колонки = месяцы Апр…Мар
• Одна таблица с колонками: Дилер, Категория, Апр, Май, …, Мар
• Один лист на категорию, название листа содержит «Оборудование» / «Расход» / «Сервис»

Загрузить отгрузки (CSV)

CSV с разделителем «;» или «,». Колонки: Дилер, Категория, Апр, Май, …, Мар. Категории: «Сервис», «Оборудование», «Расходные материалы».

Сохранить данные (JSON)

Выгрузить текущее состояние (все дилеры + факты) в JSON-файл для резервной копии.

Загрузить из JSON

Восстановить состояние из ранее сохранённого JSON-файла.

Сбросить к исходным данным

Вернуть исходных дилеров и шкалы. Все ваши изменения будут потеряны.
